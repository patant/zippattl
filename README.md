# ZippaTTL

Using Google Translate API to download MP3		
Ruby Gems:	
		
1. Sinatra	
2. Sonos 


## Installation

	cd $INSTALLATION_PATH
	bundle install
	rackup -o $IP_OF_YOURHOST

Now you should have a webserver up and running on $IP_OF_YOURHOST:9292
